# Lorraine Marange - Mavis Cares and Nurses

**Live site:** https://Lorraine-Lolo.github.io/LorraineMarange-Portfolio

## About
A responsive business website for Mavis Cares and Nurses, providing professional and compassionate home nursing care nationwide in South Africa. Originally prototyped on CodePen, then deployed via GitHub Pages.

## Built with
- HTML5
- CSS3
- JavaScript

## Features
- Company branding and logo
- About section with service details
- Contact information
- Mobile responsive design

## What I learned
- Structuring business websites with semantic HTML
- CSS styling for professional layouts
- File organization and deployment using Git and GitHub Pages
